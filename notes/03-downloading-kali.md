# 03 - Downloading Kali Linux

## Goal

Download a prebuilt Kali Linux VirtualBox image for my cybersecurity lab.

## Why I chose the VirtualBox image

Using the prebuilt image is easier for beginners because:
- Kali is already installed
- VirtualBox settings are mostly configured
- Faster setup
- Lets me focus on learning Linux and cybersecurity instead of OS installation troubleshooting

## Download Source

Official Kali Linux website:
https://www.kali.org/get-kali/

## What I learned

Virtual machines let me safely experiment with cybersecurity tools without modifying my main operating system directly.
