# 06 - New Host System Setup

## Reason

I moved the cyber lab project to a different computer and am rebuilding the lab environment.

## Host Information

### Operating System

Ubuntu

### CPU

AMD Ryzen 7 5700U with Radeon Graphics

### RAM

16 GB

## Goal

Prepare this system for VirtualBox and Kali Linux.

## Recommended VM Resources

- 4 GB RAM
- 2 CPUs
- NAT Networking
- 128 MB Video Memory

## What I Learned

Before building a virtual machine lab, it is important to understand the resources available on the host system.
