# 18 - First Packet Capture with Wireshark

## Overview

Performed my first packet capture using Wireshark on the Kali Linux virtual machine.

## Goal

Observe network traffic generated by the virtual machine and become familiar with Wireshark's interface and basic packet analysis.

## Tool Used

Wireshark 4.6.6

## Network Interface

The packet capture was performed on:

```text
eth0
```

## Activities Performed

### HTTPS Traffic Analysis

Visited websites using HTTPS and observed encrypted network traffic.

Most captured packets appeared as:

```text
TLS
Application Data
```

This demonstrated that modern web traffic is encrypted and cannot be easily read while in transit.

### HTTP Traffic Analysis

Visited:

```text
http://neverssl.com
```

and applied the following Wireshark display filter:

```text
http
```

This allowed the HTTP requests and responses to be viewed directly because the traffic was not encrypted.

## Observations

During packet capture I observed:

* IPv4 traffic
* TCP connections
* DNS requests
* TLS encrypted traffic
* HTTP web traffic

I was able to see the difference between encrypted HTTPS traffic and unencrypted HTTP traffic.

## Challenges Encountered

During the packet capture exercise, the Kali virtual machine became temporarily unresponsive. Resource monitoring showed normal CPU and memory utilization, and the VM recovered without requiring a restart.

This highlighted the importance of monitoring system resources and distinguishing between application issues and full system failures.

## What I Learned

* Every network connection is made up of packets.
* Wireshark can capture and analyze those packets.
* HTTPS encrypts data to protect information while it travels across networks.
* HTTP traffic can often be viewed directly because it is not encrypted.
* Packet analysis provides visibility into how systems communicate across a network.

## Next Steps

* Learn to identify DNS traffic
* Analyze TCP and UDP communications
* Capture ICMP (ping) traffic
* Explore additional Wireshark filtering techniques
