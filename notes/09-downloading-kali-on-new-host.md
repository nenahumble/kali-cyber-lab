# 09 - Downloading Kali Linux on the New Host

## Goal

Download a Kali Linux virtual machine image for use with VirtualBox.

## Why Kali Linux

Kali Linux is a Debian-based distribution that includes many tools used for cybersecurity, networking, and penetration testing.

## Why the VirtualBox Image

Using the prebuilt VirtualBox image allows me to focus on learning Linux and cybersecurity concepts rather than performing a full operating system installation.

## Host System

* Ubuntu 24.04.4 LTS
* AMD Ryzen 7 5700U
* 16 GB RAM

## Expected Outcome

A functioning Kali Linux virtual machine running inside VirtualBox.

## Next Steps

* Download Kali image
* Import into VirtualBox
* Configure VM resources
* Boot Kali
* Create initial snapshot
