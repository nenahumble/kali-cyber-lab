# 13 - Creating a Baseline Snapshot

## Overview

Created a snapshot of the Kali Linux virtual machine after confirming the initial installation was working correctly.

## What is a Snapshot?

A snapshot is a saved state of a virtual machine at a specific point in time. It allows the virtual machine to be restored to that exact state later if needed.

## Why I Created This Snapshot

I created a baseline snapshot to have a known good recovery point before making additional changes to the system. This provides a safe point to return to if future updates, configurations, or testing activities cause problems.

## Why Snapshots Are Important

Snapshots are important in cybersecurity labs because they allow experimentation without permanently affecting the virtual machine. If a mistake is made or a system becomes unstable, the VM can be restored to a previous working state instead of rebuilding it from scratch.

## What I Learned

Virtual machine snapshots are a valuable tool for testing, troubleshooting, and recovery. Creating snapshots before major changes can save significant time and effort when working in a lab environment.
